## About

- **Research Assistant, Boston University.** Spiking network models of neurodegenerative disease progression. [Email me](mailto:ayaan.rustagi2010@gmail.com) if you want to talk about it.
- **Research Assistant, University of Alabama.** Computational and scientific research support.
- **FBLA Texas State Champion.** Website Coding & Development, National Qualifier. District Champion in Business Ethics.
- **Secretary, RHS Science Fair Club.** Built [rhsscifair.tech](https://www.rhsscifair.tech/) for 100+ members.
- Built CITEX, Reunite, Caloric AI, four Chrome extensions, a couple of micro:bit hardware projects, and some private competition work.

### [Alarm Sentinel](https://github.com/ayaanrustagi/alarm-sentinel)
**Accessibility · micro:bit V2**
Every US smoke alarm since 1996 is legally required to beep the same rhythm. So a $15 board can listen for that NFPA 72 pattern and strobe for deaf and hard-of-hearing users, with no pairing and no ML. About 80 lines of timing logic. Commercial strobe systems run $250 to $300.

`Python` `micro:bit` `Accessibility`

</td>

### [Firefly Sync](https://github.com/ayaanrustagi/firefly-sync)
**Swarm behaviour · micro:bit**
Twenty micro:bits with no leader and no shared clock, teaching themselves to blink in unison the way fireflies do. Mirollo-Strogatz pulse-coupled oscillators, 26 tests, firmware maths asserted identical to the simulator. Simulation-verified so far.

`Python` `micro:bit` `Simulation`
